### Ejemplo micro:bit

Para comenzar con micropython en micro:bit vamos a mostrar una imagen en su pantalla:

```python
from microbit import *
print('Hola micro:bit')
display.show(Image.HAPPY)

```

Veremos en la consola el texto y en la pantalla de la micro:bit una carita sonriente.

La placa micro:bit no tiene led individuales, pero podemos usar los de la pantalla usando las 칩rdenes de la pantalla **display.set_pixel(x,y,brillo)** que veremos m치s adelante.

Por ejemplo para hacer que se active a m치ximo brillo (9) el pixel de la esquina izquierda de arriba haremos

```python
from microbit import *
display.set_pixel(0,0,9) # enciende el led a m치ximo brillo
```

y para apagarlo:

```python

```

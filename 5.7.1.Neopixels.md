## Neopixel

```python
import neopixel, machine
import Wemos

ledRGB = neopixel.NeoPixel(machine.Pin(Wemos.D2),1) 

# Rojo
ledRGB[0] = (50,0,0)
ledRGB.write()
# Verde
ledRGB[0] = (0,50,0)
ledRGB.write()
# Azul
ledRGB[0] = (0,0,50)
ledRGB.write() 
# Blanco
ledRGB[0] = (255,255,255)
ledRGB.write() 
# Negro
ledRGB[0] = (0,0,0)
ledRGB.write() 

```

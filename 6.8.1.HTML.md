## HTML


        <!DOCTYPE html>
        <html>
        <head>
        <title>Page Title</title>
        </head>
        <body>

        <h1>This is a Heading</h1>
        <p>This is a paragraph.</p>

        </body>
        </html>


## CSS

        body {
        background-color: lightblue;
        }

        h1 {
        color: white;
        text-align: center;
        }

        p {
        font-family: verdana;
        font-size: 20px;
        }
